# Ultrasonic Distance Alert System

Author: Gokulaharny  
Platform: Arduino Uno  
Sensor: HC-SR04  

## Project Overview
This project measures distance using ultrasonic waves. When an object comes within a defined threshold, a buzzer alert is triggered and the distance is displayed via Serial Monitor.

## What I Learned
- Time-of-flight measurement principles
- Converting pulse duration to distance
- Conditional alert logic

## Simple Explanation (For High School Students)
The sensor sends out sound waves and waits for them to bounce back. By timing how long it takes, the system calculates distance. If something gets too close, it makes a sound.
